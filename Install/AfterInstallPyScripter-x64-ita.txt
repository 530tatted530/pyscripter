***********  Questa è la versione a 32bit di PyScripter. *************
Se stai usando la versione di Windows a 64bit nota che PyScripter funzionerà solo se è installata una versione di Python a 64bit.

Per compatibilità con Python 2.6, 2.7 e 3.x, questa versione di PyScripter richiede una versione aggiornata del pacchetto C++ Ridistribuibile.
Questo pacchetto viene installato automaticamente durante l'installazione di Python 2.6, 2.7 e 3.x. 
Se non hai una di queste versioni installate, puoi scaricare ed installare manualmente il pacchetto da

https://www.microsoft.com/en-us/download/details.aspx?id=15336.

Visita https://github.com/pyscripter/pyscripter/wiki per informazioni sul debug remoto e altre info su PyScripter.
