## Release Checklist

1. Update history.htm and generate help file
2. Increase project version number
3. Build for x64 
4. Generate fresh pyscripter.ini and place in the Install folder (delete forms from C:\ProgramData\PyScripter)
5. Increase version number in Pyscripter.iss
6. Generate and test setup file for x64
7. Generate zip file for x64
8. Build for x86
9. Generate and test setup file for x86
10. Generate zip file for x86
11. Upload setup and zip files (4 files) to Sourceforge
12. Update PyScripterVersionInfo.ini and PyScripterVersionInfo-x64.ini
13. Blog announcement at https://pyscripter.blogspot.com/
14. Announcement at https://groups.google.com/g/pyscripter
15. Announcement at https://groups.google.com/g/comp.lang.python.announce
16. Update history at PyScripter wiki
17. Optionally create Github release
