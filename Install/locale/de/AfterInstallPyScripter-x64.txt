***********  Dies ist die 64-Bit Version von PyScripter. *************
Sie erfordert eine 64-Bit Version von Windows. Bitte beachten Sie, dass PyScripter
nur funktioniert, wenn eine 64-Bit Version von Python installiert ist.

Hinweis: zur Kompatibilität mit Python 2.6, 2.7 und 3.x, erfordert diese Version
von PyScripter das neueste C++ Redistributable Paket.  Dieses wird automatisch
von Python 2.6, 2.7 und 3.x installiert.  Wenn Sie keine dieser Versionen installiert
haben, müssen Sie das Paket manuell herunterladen und installieren:
https://www.microsoft.com/en-us/download/details.aspx?id=15336.

Bitte besuchen Sie https://github.com/pyscripter/pyscripter/wiki für Informationen
zum Remote-Debugging und andere PyScripter-spezifische Themen.
