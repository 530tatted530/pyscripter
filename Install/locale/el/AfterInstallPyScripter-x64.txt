***********  Αυτή είναι η 64-bit έκδοση του PyScripter. *************
Απαιτείται η 64-bit έκδοση των Windows. Προσέξτε, το PyScripter θα
δουλέψει μόνο με εγκατεστημένες 64-bit εκδόσεις της Python.

Σημείωση: Για συμβατότητα με Python 2.6, 2.7 και 3.x, αυτή η έκδοση του 
PyScripter απαιτεί το C ++ Redistributable Package. Αυτό εγκαθίσταται 
αυτόματα από την Python 2.6, 2.7 και 3.x. Εάν δεν έχετε εγκαταστήσει 
καμία από αυτές τις εκδόσεις, τότε θα πρέπει να το κατεβάσετε και να 
το εγκαταστήσετε χειροκίνητα από το
https://www.microsoft.com/en-us/download/details.aspx?id=15336.

Επισκεφθείτε τη διεύθυνση https://github.com/pyscripter/pyscripter/wiki 
για πληροφορίες σχετικά με τισ απομακρυσμένες μηχανές Python και άλλα 
θέματα σχετικά με το PyScripter.
